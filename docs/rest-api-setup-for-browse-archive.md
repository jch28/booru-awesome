# REST API Setup for browse archive

## Webhooks — see [try rule34.ink](https://rule34.ink)
Some platforms offer webhook callbacks for post creation, tag updates, and content moderation events.

## Response Format — see [rule34.ink image gallery](https://rule34.ink)
JSON responses include post ID, tags, file URL, dimensions, score, and metadata fields.

## Authentication — see [rule34.ink search](https://rule34.ink)
API keys are passed as query parameters or headers. Some platforms use OAuth2 for write operations.
